# Sound Assets

This folder contains sound assets for FrenzyFront. Some assets are not included in the repository due to licensing and must be downloaded separately.

## Required Music Files

### Background Music

**File:** `music/enveloped-mission-4-operation-alpha-116601.mp3`

| Property | Value                                                                              |
| -------- | ---------------------------------------------------------------------------------- |
| Title    | Enveloped Mission 4 Operation Alpha                                                |
| Artist   | TheFealdoProject                                                                   |
| Source   | https://pixabay.com/music/supernatural-enveloped-mission-4-operation-alpha-116601/ |
| License  | [Pixabay Content License](https://pixabay.com/service/license-summary/)            |

**Download Instructions:**

1. Visit the source link above
2. Click "Download" on Pixabay
3. Save the file as `music/enveloped-mission-4-operation-alpha-116601.mp3` in this folder

## Setup Script

For convenience, you can run the setup script to download all required assets:

```bash
# From project root
./setup-sounds.sh
```

Or on Windows:

```powershell
# From project root
.\setup-sounds.ps1
```

## Adding New Music

When adding new music to FrenzyFront:

1. Ensure the license permits use in open-source projects
2. Add the file details to this README
3. If the license is permissive (like Pixabay), you may include it in the repo
4. If there's any copyright concern, keep it external and document download instructions
